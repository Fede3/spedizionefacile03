# Utility Codex locali

Questa cartella raccoglie utility locali per l'ambiente Codex, separate dai
launcher operativi del progetto.

## Contenuto

- `AVVIA_CODEX.bat` - apre Codex in Git Bash dalla root repository
- `INSTALLA_CODEX_WSL.bat` - setup guidato WSL/Ubuntu per Codex

Questi file non fanno parte del runtime del sito e restano qui per tenere la
repo piu' chiara e con ownership piu' netta.
