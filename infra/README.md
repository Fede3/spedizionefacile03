# Infra

Questa cartella contiene solo infrastruttura e configurazioni operative condivise.

Canonico:
- `caddy/` per la configurazione Caddy attuale

Non mettere qui:
- documentazione di prodotto
- log locali
- screenshot o prove temporanee
- script di sviluppo generici che non sono infrastruttura

Se aggiungi una nuova sottocartella, deve avere ownership chiara e un README locale se non è autoesplicativa.
