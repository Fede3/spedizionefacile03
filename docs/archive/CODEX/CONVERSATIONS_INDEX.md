## COME LEGGERE I FILE JSONL

Ogni file JSONL è una conversazione. Ogni riga è un JSON con struttura:
```json
{"type": "user", "message": {...}}
{"type": "assistant", "message": {...}}
```

Per leggere un file (i file grandi richiedono offset/limit):
```
Read file: C:\Users\Feder\.claude\projects\...\session.jsonl
Usa offset e limit per navigare le sezioni
```

Per estrarre solo i messaggi utente (testo puro):
```bash
python -c "
import json, sys
for line in open('file.jsonl'):
    try:
        obj = json.loads(line)
        if obj.get('type') == 'user':
            content = obj.get('message', {}).get('content', '')
            if isinstance(content, str):
                print('USER:', content[:200])
            elif isinstance(content, list):
                for c in content:
                    if isinstance(c, dict) and c.get('type') == 'text':
                        print('USER:', c.get('text', '')[:200])
    except: pass
"
```
